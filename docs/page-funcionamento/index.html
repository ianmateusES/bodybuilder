Dashboard

Tela de inicio é a que vem logo após o login do usuário. Nela podemos ver se concentram todas as funcionalidades do sistema no lado esquerdo. Em seu meio temos a listagem de alunos ativos e inativos do personal. Podendo o personal fazer:

  • Visualizar os aniversariantes do dia
  • Atualizar a lista de alunos
  • Abrir as informações dos alunos, listando os treinos
  • Pesquisar por nome do aluno

Agenda

Nessa funcionalidade, o personal podera acompanhar sua agenda de consultas marcadas. Podendo o personal fazer:

  • Pesquisar por nome do aluno e/ou por data
  • Atualizar a tabela de alunos
  • Cadastrar uma nova consulta

Listagem de exercícios

A tela de listagem de exercícios, mostra os exercicios cadastrados pelo personal para a utilização no desenvolvimento de treinos. Podendo o personal fazer:

  • Filtrar pelos exercícios do sistema e/ou cadastrado pelo personal
  • Atualizar a tabela de exercicio
  • Pesquisar pelo nome do exercícios ou pelo grupo que ele pertence
  • Cadastrar um novo exercício

Listagem de alunos

Funcionalidades similar a da tela de Dashboard. Mas com uma visão mais ampla das informações do aluno. Podendo o personal fazer:

  • Filtrar por alunos ativos e/ou inativos
  • Cadastrar novos alunos
  • Pesquisar pelo nome do aluno
  • Atualizar a tabela de alunos
